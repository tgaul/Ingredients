enabled

A multiple-value binding that determines if the NSSlider is enabled in the user interface.

When enabled is bound, a new binding, enabled2, is exposed and can be bound. Binding to enabled2 causes enabled3 to be exposed, and so on.

The NSSlider is enabled if a logical AND operation on all the enabled bindings results in a Boolean value of YES.

Availability:
Available in Mac OS X v10.3 and later.
Binding is Read-Only.
